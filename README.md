# NULLSTEP

software that predicts hesitation.

---

## what is this?

most software waits.

nullstep doesn’t.

it watches the space between intention and action —  
the cursor pause, the backspace, the almost-click.

and then it responds.

---

## current state

this is not stable.

this is not optimized.

this is being built in public.
